# UIGen Project Standards

## Always apply these rules
- Use TypeScript strict mode
- Use Tailwind CSS for all styling — no inline styles
- Use neutral-* instead of gray-* for colors
- Never modify the database schema without asking first
- Never expose API keys or sensitive data

## Tech Stack
- Next.js 15, React 19, TypeScript
- Prisma with SQLite
- Tailwind CSS
- Anthropic API via Vercel AI SDK

## Code Style
- Clean, readable components
- Split complex UIs into multiple files
- Always handle loading and error states
